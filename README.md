# java-cli-maven-html-to-pdf

## Description
A demo of html rendering to pdf. In this demo
we can use flying saucer which is based on itext library.
This will not render css grid.

## Tech stack
- flying saucer
- maven

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
